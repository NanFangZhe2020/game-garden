#### 技术介绍
css + jq
#### 游戏名称
《我的数独我做主》
#### 游戏规则
随心所欲，自己的数独自己做主；想怎么分配数字就怎么分配数字。（没有限制，任由发挥，可以创建空的数独，再去自己填补完整的数独。也可以直接创建完整的数独，再点击解答~）
1. 建数独
输入数字到对应位置，创建数独。
2. 解数独
完成数独创建，再去解数独。
3. 完成
流程完成会出现有显示创建的完成时间、解答的完成时间~
